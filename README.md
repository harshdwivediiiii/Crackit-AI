<p>To set up and run the project locally, follow these steps:</p>

<h3>1. Clone the Repository</h3>
<pre><code>git clone https://github.com/your-username/crackit-ai.git
cd crackit-ai</code></pre>

<h3>2. Install Dependencies</h3>
<pre><code>npm install
# or
yarn install
# or
pnpm install
</code></pre>

<h3>3. Set Up Environment Variables</h3>
<p>Create a <code>.env.local</code> file in the root directory and add the following variables:</p>
<pre><code>NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your-clerk-publishable-key
CLERK_SECRET_KEY=your-clerk-secret-key
NEXT_PUBLIC_DRIZZLE_DB_URL=your-neon-database-url
NEXT_PUBLIC_GEMINI_API_KEY=your-gemini-api-key
</code></pre>

<h3>4. Run Development Server</h3>
<pre><code>npm run dev
# or
yarn dev
# or
pnpm dev
</code></pre>

<p>Open <a href="http://localhost:3000" target="_blank">http://localhost:3000</a> with your browser to see the site in action.</p>
